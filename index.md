## PacketEvents API

### What is PacketEvents?
PacketEvents is an open source Spigot Packet API supporting 1.7.10 to the latest spigot version. 
The API is updated frequently with new optimizations and features.
PacketEvents is licensed under the MIT license.
We offer quick support on our discord server!
